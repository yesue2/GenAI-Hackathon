# 2023 관광분야 GEN AI 해커톤
**한국관광공사 주관**

## 💻 프로젝트 주제
Chat GPT를 활용한 **개인 성향 맞춤 여행코스 추천** 애플리케이션

## 📆 개발 기간
2023.08.03 ~ 07

## 👨‍👨‍👧‍👦 팀 구성
기획1, 안드로이드2, 백엔드1, 디자인1

## 📌 주요 기능
1. **Big Five Model기반**으로 **사용자의 성격유형을 분석**
2. **생성형 인공지능(Chat GPT)을 활용**하여 **사용자 성향 맞춤 여행 코스 분석** 및 다음 여행지 추천
